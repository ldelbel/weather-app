## Getting Started

To get a local copy up and running follow these simple example steps.

```
$ cd <folder>
$ git clone https://github.com/ldelbel/weather-app
$ cd weather-app

```

### Setup

Setup the environment for webpack.

```
$ npm install
$ npm run build
$ npm start  // for live view

```

### Prerequisites

- npm
